A beautiful and functional music player! 🎶  
I developed this project while thinking about how to create a music player **without using any external platform**.  
You can play your favorite music anytime without needing to stay on the site.  
Enjoy listening while continuing your work on **desktop or mobile**.  

🔊 **Enjoy the music!**

## 🚀 Features  
- Easy-to-use and stylish interface 🎨  
- Background play support 🔄  
- Compatible with **desktop and mobile** 📱💻  
- Lightweight and fast performance ⚡
